# 📋 Smart Task Manager

A modern, responsive **Task Management Web App** built using HTML, CSS, and JavaScript.
This application helps users efficiently manage tasks with advanced features like scheduling, filtering, drag-and-drop, and analytics.
---
## 🌐 Live Demo
👉 https://smart-task-manager-wheat.vercel.app/
---

## ✨ Features
### 🧩 Core Features
* Add, Edit, Delete Tasks (Full CRUD)
* Set **Date & Time** for tasks
* Mark tasks as Completed / Pending
* Priority Levels (High, Medium, Low)
* Category-based organization
  
### 🔍 Smart Features
* Search tasks instantly
* Filter by:
  * Status (Completed / Pending)
  * Priority
* Real-time countdown timer for each task
* Overdue task highlighting

### 🚀 Advanced Features
* 🧲 Drag & Drop task reordering
* 🔔 Browser Notifications (task reminders)
* 📊 Analytics Dashboard (Completed vs Pending chart)
* 🌙 Dark Mode support
* 💾 LocalStorage (data saved even after refresh)
---

## 🛠️ Tech Stack
* HTML5
* CSS3 (Glassmorphism UI + Responsive Design)
* JavaScript (Vanilla)
* Chart.js (Analytics Visualization)
* LocalStorage API
